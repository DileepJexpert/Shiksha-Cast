# Experiment Plan - Sach Ki Kahani

## Category To Test

Start with **social-awareness short drama**:
- 45-60 seconds
- Hindi/Hinglish
- one corruption/social issue
- first line has shock/conflict
- final line asks viewers what they would do

This is a better discovery experiment than slow tutorials because the viewer gets
an immediate emotional question.

Track:
- Shorts viewed vs swiped away
- average percentage viewed

- thumbnail CTR for horizontal upload
