# Sach Ki Kahani - Social Short Drama Format

Use this category when views are low and we need stronger viewer curiosity.

## Format

- Length: 35-60 seconds
- Language: Hindi/Hinglish
- Style: fictional social-awareness drama
- Structure: hook -> injustice -> proof -> action -> viewer question
- Best platform test: YouTube Shorts first, then horizontal version if it gets comments

## Script Formula

1. First line: shocking but believable issue.
   Example: "Exam se pehle paper pachaas hazaar mein bik raha tha."
2. Emotional victim line.
   Example: "Maine imandari se padhai ki thi."
3. Pressure line from corrupt side.
   Example: "Chup raho. System aise hi chalta hai."
4. Proof/reveal line.
   Example: "Mere paas chat, payment number, aur file photo hai."
5. Action line.
   Example: "Investigation abhi shuru hoti hai."
6. Comment question.
   Example: "Aap chup rahenge ya awaaz uthayenge?"

## Topic List

- Paper leak / exam scam
- Ration corruption
- Hospital medicine theft
- Road contractor scam
- Bribe for certificate
- Police complaint bribe
- Fake job list
- School scholarship theft
- Water tanker mafia
- Local office file delay

## Thumbnail Rule

Use 3 elements only:
- one victim face
- one corrupt/problem figure
- one big text promise

Example: `PAPER LEAK - 50,000 MEIN PAPER?`
