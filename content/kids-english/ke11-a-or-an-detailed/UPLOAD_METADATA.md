## Title
A or An for Kids | Beginner English Grammar with Kinnu | Class 1-3

## Description
Learn A and An slowly with Kinnu in a friendly classroom lesson.

This video teaches the beginner rule:
- Use AN before A, E, I, O, U.
- Use A before other letters.

Kids practise simple examples:
- an apple
- an egg
- an ice cream
- a ball
- a cat
- a dog
- a mango
- a kite
- a school

Note for parents and teachers: this is a beginner Class 1-3 rule. Later lessons can teach sound exceptions like "an hour" and "a university".
