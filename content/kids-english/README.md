# Kids English Micro-Lessons

Small board-style English lessons for kids. Each lesson is designed as a short slide presentation first, so it can be used for:

- YouTube slideshow videos
- Shorts cutdowns
- classroom revision
- future animated host overlay

Format:

- 8 slides per topic
- green classroom board style
- small Kinnu teacher sticker on each slide
- one simple rule per lesson
- examples, practice, recap
- deterministic PNG rendering, no AI text mistakes

Starter sequence:

1. `ke01-a-or-an` - choosing `a` and `an`
2. `ke02-this-that` - near and far objects
3. `ke03-am-is-are` - simple helping verbs
4. `ke04-he-she-it` - pronouns for one person/thing
5. `ke05-i-you-we-they` - pronouns for self, listener, group
6. `ke06-has-have` - possession words
7. `ke07-do-does` - question helpers
8. `ke08-was-were` - past tense helpers
9. `ke09-can-cannot` - ability words
10. `ke10-there-is-there-are` - one and many things
