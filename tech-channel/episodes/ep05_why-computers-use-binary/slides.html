Idea 1 / 4 — It's all switches

Inside: billions of tiny switches.

🔌 OFF= 0
⚡ ON= 1

These switches are called transistors.

  • A computer is built from billions of microscopic switches.
  • Each one is either OFF or ON — two states.
  • OFF = 0, ON = 1. That's the whole alphabet.
Katixo KhojLab
Idea 2 / 4 — Why two? Because it's reliable

Two states are easy to tell apart.

  • Like a light switch — clearly on or off (high or low voltage).
  • Ten different levels? Easy to confuse → errors.
  • Simple = reliable = fast. This is the key insight.
2 levels

Hard to mistake ON for OFF.

10 levels

Is that a 6 or a 7? Too risky.

Katixo KhojLab
Idea 3 / 4 — Line them up to count

One switch is small. Many can count anything.

# one switch = 1 bit = 2 options

# line up 4 bits → read like place value
 8   4   2   1
 1   0   1   0   →  8 + 2 = 10
  • Each 0/1 is a bit.
  • 8 bits = 1 byte = 256 combinations.
  • Enough bits → any number you want.
Katixo KhojLab
Idea 4 / 4 — Everything becomes a number

Numbers → binary. So binary stores it all.

🔤 letters→ a code (ASCII / Unicode)
🎨 colours→ Red, Green, Blue values
🔊 sound→ thousands of samples
  • First turn it into numbers…
  • …then every number becomes binary.
  • One simple system stores everything.

Why it matters

This is why "digital" is so reliable.

  • Two clean states is why computers rarely make mistakes.
  • Simple switches can be made tiny — billions on one chip.
  • Tiny + simple = cheap and fast. That's the whole digital world.

✗ Myth

"Binary is some weird, secret code that only computers understand."

✓ Reality

It's just counting with two symbols — because the hardware physically has two states: on / off switches. Nothing mysterious.
