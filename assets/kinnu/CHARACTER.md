# Kinnu Character Reference

Use `kinnu-master-reference.png` as the canonical Kinnu design.

Style lock:
- Cute friendly girl stickman mascot for a kids learning channel.
- Big round white head, big expressive eyes, warm smile.
- Short dark hair with a pink bow.
- Yellow top, blue pants, white shoes, thick black outlines.
- Bright flat cartoon style, wholesome and kid-safe.
- Future Kinnu images should resemble this design, not the original photo.

Notes:
- `kinnu-inspired-reference.png` is the first generated concept.
- `kinnu-master-reference.png` is the stable reference filename for prompts, thumbnails, pose sheets, and animation assets.
- The original HEIC photo is not part of the repo and should only be treated as loose inspiration.
