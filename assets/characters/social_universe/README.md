# Social Universe Characters

Use this folder for realistic or semi-realistic characters used in social-awareness
stories, corruption stories, public-service stories, and adult/family dramas.

Suggested ids:

- `politician_hd`
- `journalist_hd`
- `officer_hd`
- `clerk_hd`
- `teacher_hd`
- `grandmother_hd`
- `student_hd`
- `common_man_hd`

In a story config, refer to these as `social_universe/politician_hd`,
`social_universe/journalist_hd`, and so on.
