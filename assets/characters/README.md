# Character Asset Universes

Use this folder for new reusable character libraries.

- `kinnu_universe/` - Kinnu, Vibhu, Gappu, Buddy, and other kids-channel cartoon characters.
- `social_universe/` - realistic or semi-realistic public-awareness/story characters.

Legacy rigs under `assets/cartoon/characters/` still work for older videos, but new
characters should be placed here to avoid mixing cartoon and realistic casts.
